What's in these folders?

    code:   Source code.

    learn:  Overview, APIs (Syntax), and FAQ (Help) pages.

    play:   API Examples, Demos, and Ideas.

    test:   Unit Tests.

    use:    ajile's minified default and versioned scripts
            + ajile's versioned source
            + other scripts and resources used by pages in /learn and /play. 

    index.htm
        : Starting point that redirects to learn/Ajile-Overview.htm


What do I need to use ajile in my app or on my site?

    use/index.js and any use/com.iskitz.ajile script; nothing else.
